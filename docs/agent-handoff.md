# Agent Handoff Protocol

## 原則

1. AIが進められる作業は止めない。
2. Secretsの実値は書かない。
3. 失敗したらログ、原因、次の一手を残す。
4. 人間が必要な場合は、必要操作を最小単位で明記する。
5. Codex / Claude Code / Gemini の役割を分け、次のAIがすぐ再開できる形にする。

## Agent別の担当

### Codex

- 実装
- バグ修正
- テスト作成
- CI失敗修正
- PR差分整理

### Claude Code

- 設計
- docs整備
- README更新
- 初期設定手順の分解
- リファクタリング方針整理

### Gemini

- 調査
- QA
- 外部API比較
- ログ検証
- 再現手順確認

## 引き継ぎメモに必ず残す項目

- 何をしようとしたか
- どこまで完了したか
- 失敗した場合のログ
- 次に実行すべきコマンド
- 必要なSecret名
- 人間が必要な場合の理由

## 完了条件

- テストが通る
- CIが通る
- README/docsが更新される
- 未完了タスクがNotion / Google To Do / Artifactに残る
- 人間が必要な作業が最小化されている
